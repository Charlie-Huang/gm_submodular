L1 contraint learning of submodular mixtures

Method

In this work, we propose a novel formulation for learning submodular mixture weights. It is an extension of [1], where the problem is formulated as:

$\underset{\mathbf{w}\geq 0}{\min}\frac{1}{T}\sum_{t=1}^{T}\hat{L}_t(\mathbf{w}) + \frac{\lambda}{2}||\mathbf{w}||^2.$

Rather than minimizing the $l_2$-norm as [1], we propose an $l_1$-constraint minimization, i.e.

$ \underset{\mathbf{w}\geq 0}{\min}\frac{1}{T}\sum_{t=1}^{T}\hat{L}_t(\mathbf{w}) \\ \text{s.t.} \sum_{i=1}^{n}w_i \leq 1, w_i \geq 0 \forall i $

where the loss function is defined as [1]:

$ \hat{L}_t(\mathbf{w}) = \underset{\mathbf{y}\subseteq \mathbf{Y_V}^{(t)}}{\max} ( \mathbf{w}^\mathrm{T}\mathbf{f}(\mathbf{x}_\mathbf{V}^{(t)},\mathbf{y}) + l_t(\mathbf{y}) ) - \mathbf{w}^\mathrm{T}\mathbf{f}(\mathbf{x}_\mathbf{V}^{(t)},\mathbf{y}_\text{gt}^{(t)}), $

In order to minimize this problem, we follow [1] and use stochastic subgradient descent. In [1], the $w_t$ is projected back into the feasable set using $\mathbf{w_t}=max(\mathbf{0,w_{t-1}}-\eta_t\mathbf{g_t})$. Instead, we project it back into the $l_1$-semiball, by solving the following optimization problem:

$ \mathbf{w_t}=\underset{\mathbf{w_t}}{\arg\min}\left(||\mathbf{w_t}-(\mathbf{w_{t-1}}-\eta_t\mathbf{g_t})||^2_2\right) \\\text{s.t.} \sum_{i=1}^{n}w_i \leq 1, w_i \geq 0 \forall i %FIXME: currently we normalize the wegihts to sum to one afterwards: w[it]=w[it]/np.sum(w[it]). remove? or use it as a constraint in the optimization? $

By projecting the learnt weights into the $l_1$-semiball, our approach learns sparser weights, which makes it more robust to noise. In the following we present a set of simple experiments, comparing the proposed method to [1].

[1] Lin, H. & Bilmes, J. Learning mixtures of submodular shells with application to document summarization. UAI 2012

[2] Leskovec, J., Krause, A., Guestrin, C., Faloutsos, C., VanBriesen, J., & Glance, N. Cost-effective outbreak detection in networks. ACM SIGKDD 2007

[3] Gygli, M., Grabner, H., & Gool, L. Van. Video Summarization by Learning Submodular Mixtures of Objectives. CVPR 2015

Experiments

We evaluate our method on a simple toy dataset. Thereby we proceed as follows:

  1. We create a dataset of random points in 2D (see Figure below).
  2. We define a set of submodular objectives each with a predefined weight.
  3. Obtain ground truth by optimizing the objectives using [2].
Given this data, we use our own and the method of [1] to reconstruct/learn the weights. The error in reconstructing the initial weights serves as a performance metric. As a loss function we use a 1-minus intersection loss:

$ l_t(\mathbf{y}) = \frac{1}{B}\left(\left\vert{ \mathbf{y} }\right\vert - \left\vert{ \mathbf{y} \cap \mathbf{y}_\text{gt}^{(t)} }\right\vert\right) $

i.e. it is a count of how many of the candidate summary y are not represented in the ground truth, normalized by the maximal length of the summary [3].

Objectives

We use two objectives, a k-medoid scoring [3] and a simple modular objective, which uses sum of x coordinates of a solution (examples: see below).

As a noise objective, we further assign each datapoint a random score and use the sum over these scores.
